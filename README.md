# Growth Tracker

用户增长实验追踪与渠道分析 CLI 工具 - 帮助你科学地测试和优化获客渠道。

## 核心功能

- ✅ 记录多渠道获客实验（渠道名、投入成本、获客数、转化率）
- 📊 自动计算 CAC（获客成本）和 ROI
- 📈 生成渠道效果对比报告（Markdown/CSV 导出）
- 💡 内置 10+ 低成本获客策略清单与执行模板

## 数据说明

- **CAC (Customer Acquisition Cost)**: 获客成本 = 总投入 / 获客数
- **转化率**: 转化数 / 获客数 × 100%
- **ROI**: (收入 - 成本) / 成本 × 100%（假设每个转化用户价值 ¥100）

## 最佳实践

1. **同时测试 3-5 个渠道** - 不要把鸡蛋放在一个篮子里
2. **每个渠道至少坚持 2 周** - 给渠道足够的测试时间
3. **记录详细备注** - 记录每次实验的具体做法，方便复盘
4. **定期生成报告** - 每周查看数据，及时调整策略
5. **砍掉低效渠道** - ROI 最低的渠道果断放弃
6. **加倍投入高效渠道** - 把资源集中在最有效的渠道
